# portfolio assignment website 

A simple, responsive personal portfolio website built using HTML, CSS, and JavaScript.  
It includes key sections (About, Projects, Skills, Contact) and interactive features like theme toggle, smooth scrolling, and form handling.


## Features

- Responsive Design (desktop / tablet / mobile) using CSS Grid + media queries  
- Dark/Light Theme Toggle with saved preference using `localStorage  
- Smooth Scrolling Navigation to page sections  
- Contact Form Interactivity (validation + success message)
