Technical Deep Dive

Challenges & Solutions

Problem

Duplicate CSS rules from previous assignments caused dark mode specificity conflicts

Solution

Audited & removed ~150 duplicate lines; consolidated all tokens into CSS custom properties

Problem

Mobile timeline layout — the connecting dot floated away from the card on small screens

Solution

Collapsed to single-column at 768px; fixed left/padding values for the dot in the media query

Unresolved

Contact form doesn't send real emails — submission is still simulated client-side only

Planned

Integrate EmailJS or Formspree for real delivery without a backend in a future update
