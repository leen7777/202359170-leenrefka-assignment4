Technical Deep Dive — Resolved
08 / 12

Technical Deep Dive

Challenges Resolved

Problem

~150 lines of duplicate CSS from previous assignments caused dark mode specificity conflicts — some elements ignored the theme variable.

Solution

Full CSS audit: consolidated all colour, spacing, and animation tokens into a single set of CSS custom properties. Removed every duplicate block.

Problem

x

Mobile timeline layout — the connecting dot floated away from the card on screens below 768px due to absolute positioning conflicts.

Solution

x

Collapsed timeline to single column at 768px breakpoint; overrode left, padding-left, and text-align values explicitly in the media query.

Problem

x

Hamburger outside-click handler used mousedown which fired before the button's click event on touch devices, causing the menu to immediately close after opening.

Solution

x

Switched event listener from mousedown to click. Added a guard to skip the handler if the click target is the hamburger button itself.

Problem

x

Contact form submit button had no id attribute — the JS referenced #submitBtn but the element was unreachable, silently failing.

Solution

x

Added id="submitBtn" and id="submitText" to the HTML. Verified with DevTools that the loading state and success message now trigger correctly.

Technical Deep Dive — Innovation & Lessons
10 / 12

Technical Deep Dive

Innovation & Lessons Learned

Innovative Solutions

💬

Rule-based chatbot

14-topic knowledge base built from my CV — no API costs, works offline, answers questions recruiters actually ask.


🎲

3D card tilt

Mouse-tracked perspective transform using only mousemove — creates depth without any 3D library.


🎉

Canvas confetti

Hand-written particle system on an HTML canvas — 120 pieces, per-particle drift, rotation, and auto-cleanup after 3 seconds.


📊

IntersectionObserver everywhere

Used for scroll reveal, skill bars, and stat counters — no scroll event polling, no performance cost.

Lessons Learned

🧠

IntersectionObserver is powerful

Replaces scroll listeners for most animation triggers. Cleaner, more performant, and the threshold option gives precise control.


🎨

CSS variables are a design system

Defining all tokens in :root meant the entire theme switch was one attribute change — no JavaScript needed for colours.


♿

Accessibility is architecture

aria-expanded, aria-live, and role aren't optional extras — they change how assistive tech interprets the page and should be planned from the start.


🔍

Review AI output critically

Every generated snippet needed testing and adaptation. AI accelerates — it doesn't replace — understanding the code.
