# Presentation Files

Place your presentation assets in this folder:

- `slides.pdf` — your 5–7 minute presentation slide deck
- `demo-video.mp4` — recorded walkthrough of the live application

See the assignment guidelines for the required presentation structure:
1. Introduction (1 min)
2. Technical Demonstration (3–4 min)
3. Technical Deep Dive (1–2 min)
4. Conclusion (1 min)
